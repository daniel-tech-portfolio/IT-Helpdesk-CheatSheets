# IT Helpdesk Troubleshooting Cheat Sheets

This repository contains practical troubleshooting guides designed for entry-level IT support and helpdesk technicians.

The goal of this repository is to demonstrate structured troubleshooting methods used in real IT support environments.

These documents outline common issues, diagnostic steps, and potential solutions that technicians encounter when supporting users.

---

# Topics Covered

### Password Reset Procedures
Guidelines for securely resetting user passwords while verifying identity and maintaining security best practices.

Skills demonstrated:
- Account recovery workflow
- Identity verification
- Security awareness

---

### Printer Troubleshooting
Step-by-step process for diagnosing common printer issues including connectivity problems, driver issues, and print queue errors.

Skills demonstrated:
- Hardware troubleshooting
- Network printer support
- Driver diagnostics

---

### Network Connectivity Checklist
Structured workflow for diagnosing network connectivity problems.

Topics include:
- IP configuration checks
- DNS troubleshooting
- Gateway testing
- Ping and traceroute usage

Skills demonstrated:
- TCP/IP troubleshooting
- DNS diagnostics
- Connectivity testing

---

### Active Directory User Support
Common Active Directory tasks performed by IT support technicians.

Examples include:
- Password resets
- Unlocking accounts
- Group membership checks
- User property management

Skills demonstrated:
- Identity management
- Domain administration basics
- User support operations

---

### Ticket Triage Workflow
Basic process for evaluating, prioritizing, and escalating helpdesk tickets.

Skills demonstrated:
- Incident prioritization
- Documentation practices
- Escalation procedures
